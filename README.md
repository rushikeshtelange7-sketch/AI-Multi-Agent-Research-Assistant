🤖 AI-Powered Multi-Agent Research Assistant

Overview

An AI-powered research assistant that uses multiple AI agents to generate research reports automatically.

Features

- 📋 Planner Agent
- 🌐 Research Agent
- 🧠 Analyst Agent
- ✍️ Writer Agent
- ✅ Fact Checker Agent
- 📄 Download TXT Report
- 📄 Download PDF Report
- 🌐 Streamlit Web Interface

Technologies Used

- Python
- Streamlit
- Ollama
- Llama 3.2
- DDGS
- LangChain
- FPDF
